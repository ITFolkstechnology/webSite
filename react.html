Mid-Level React/React Native Developer

We are seeking a talented and experienced React/React Native Developer to join our development team. The successful candidate will be responsible for developing and maintaining web and mobile applications using React and React Native. The candidate should also have experience with Redux or other state management libraries, as well as proficiency in writing unit tests.

Responsibilities:

- Develop, test, and maintain high-quality web and mobile applications using React and React Native.
- Collaborate with cross-functional teams to design, develop, and implement new features.
- Optimize applications for maximum speed and scalability.
- Write clean, efficient, and well-documented code.
- Stay up-to-date with emerging trends and technologies in web and mobile development.
- Implement and maintain state management using Redux or other similar libraries.
- Write unit tests to ensure code quality and prevent regressions.

Requirements:

- Proven experience in software development, with a focus on React and React Native.
- Bachelor's degree in Computer Science, Software Engineering, or related field (additional requirement).
- Strong proficiency in JavaScript, HTML, CSS, and related web technologies.
- Familiarity with RESTful APIs and modern front-end build pipelines and tools.
- Experience with Redux or other state management libraries.
- Proficiency in writing unit tests using tools like Jest, Enzyme, or React Testing Library.
- Familiarity with Git or other version control systems.
- Ability to work independently and collaboratively in a team environment.
- Excellent problem-solving skills and attention to detail.
